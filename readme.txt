FRENCH / FRANCAIS :

Voici le jeu du juste prix !

Le but de ce jeu est extrèmement simple ! Le programme génére un chiffre entre 1 et 1000 et vous devez le deviner.

Voici comment lancer le jeu :

1) téléchargez le repos
2) rendez vous dans le dossier ou se trouve le jeu
3) lancez un cmd
4) tapez "python just_price_game.py" ou "py just_price_game.py" cela dépend de la version de python

BON JEU A TOUS !!!
